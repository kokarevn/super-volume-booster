# Chrome Web Store Listing Draft

## Short description

Free privacy-first volume booster and equalizer for Chrome. Boost tab audio up to 600% with no tracking, no analytics, and no remote servers.

## Full description

**Super Volume Booster & Equalizer** is a free, open-source Chrome extension that helps you boost and tune audio in the active browser tab.

It is built for users who want a simple audio tool without suspicious permissions, tracking scripts, analytics, or hidden data collection.

### Features

- Boost tab audio up to 600%.
- Tune bass, mids, and treble with a 3-band equalizer.
- Use built-in presets: Bass+, Treble+, Rock, Flat.
- Save settings locally on your device.
- Process audio locally using the Web Audio API.
- No accounts, no ads, no analytics, no tracking.

### Privacy

The extension does not collect browsing history, page content, audio recordings, or personal data. Audio processing happens locally in the browser.

### Permissions explanation

- `activeTab`: allows the extension to work with the tab the user activates.
- `tabCapture`: captures tab audio for local processing.
- `storage`: stores volume and equalizer settings locally.
- `offscreen`: enables Manifest V3-compatible background audio processing.
